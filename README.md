<h1>Itin</h1>

<p>
Itin is a web app for creating shared travel itineraries.<br>
Users can create itineraries and add events.</p>

Functional Requirements:
<ol>
<li>Create new itinerary
<li>Delete itinerary
<li>Create Event
<li>Edit Event
<li>Delete Event
</ol>

Models:
<ul>
<li>Itinerary
<li>Events
</ul>
